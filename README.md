# Tienda Enjoy

Tienda Enjoy is an online shop for gamers created with ReacJs for Coderhouse ReactJs course.


## Run Locally

Clone the project

```bash
  git clone https://github.com/Lucascis/tienda-enjoy
```

Go to the project directory

```bash
  cd tienda-enjoy
```

Install required dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
